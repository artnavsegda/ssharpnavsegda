The SIMPL#Pro.zip package is intended for Crestron Certified Drivers SIMPL#Pro users. It contains the following folders:

Libraries -> All of the CCD dlls required to create drivers, as well as use those drivers on Crestron control systems.
Documentation -> Any SDK reference documentation.
Sample Programs -> SIMPL#Pro CCD sample projects for each device type.

Getting Started
====================
1. Run MasterInstaller and update software to the latest versions.

2. Install the latest SIMPL#Pro plugin from the Crestron website.

3. Read "Crestron RCertified Drivers - Overview.pdf" to understand the interfaces and CCD driver model.
